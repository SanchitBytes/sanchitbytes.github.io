# sanchitbytes.github.io

Personal portfolio site for **Sanchit Gupta** — AI/ML Engineer.

🔗 **Live:** [sanchitbytes.github.io](https://sanchitbytes.github.io)

## Stack

- Vanilla HTML + CSS + JavaScript (no framework, no build step)
- Hosted on GitHub Pages
- Google Fonts: Inter + JetBrains Mono

## Features

- Dark AI engineer theme with glassmorphism cards
- Canvas neural-network particle animation
- Typewriter hero subtitle
- Scroll-reveal animations (IntersectionObserver)
- Sticky frosted-glass nav
- Fully responsive + accessible (`prefers-reduced-motion` respected)
- SEO: Open Graph, Twitter Card, JSON-LD schema, sitemap

## Structure

```
sanchitbytes.github.io/
├── index.html        # entire site (HTML + embedded CSS + JS)
├── Profile-1.pdf     # resume (linked from Download Resume button)
├── robots.txt
├── sitemap.xml
└── images/
    ├── profile.JPG
    └── favicon.ico
```
